# eslint-config
custom eslint config for javascript project


## eslint-config-alziqziq
eslint config for javascript basic
- airbnb
- prettier
- jsdoc


### installation
```shell
yarn add eslint-config-alziqziq --dev
```

### usage
.eslintrc.js
```js
{
  extends: "alziqziq"
}
```
